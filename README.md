## 🛠️ Getting Started

### Prerequisites

To run the codes in this repository, you'll need a C++ compiler (like GCC/MinGW) installed on your system.

### Compilation & Execution

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vaibhav2118/CPP-Practice.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd CPP-Practice
   ```

3. **Compile and Run (e.g., Hello.cpp):**
   ```bash
   g++ Hello.cpp -o Hello
   ./Hello
   ```
